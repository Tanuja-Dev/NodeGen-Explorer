**🌳 NodeGen Explorer: Hierarchical Search Visualizer**

An interactive platform designed to visualize the mechanics of search algorithms within complex, user-defined tree structures.


**💡 Project Idea**

Abstract search algorithms can be difficult to grasp through static code alone. NodeGen Explorer bridges this gap by providing a dynamic, web-based environment. It transforms theoretical search logic into a visual experience, allowing users to watch how an algorithm "thinks" as it navigates through parent-child relationships in real-time.


**🚀 Features**

🎯 Dynamic Tree Generation
🔍 Depth-Limited Search (DLS)
⚙️ Real-time Constraint Tuning
🚫 Clean Interface
📈 Path Tracing


**🧠 AI Concepts Used**

**Depth-Limited Search (DLS):**
The core traversal logic used to explore nodes up to a specific, user-defined threshold to handle the "state space explosion" problem.

**Uninformed Search Strategies:**
Implementation of Depth-First principles to demonstrate the fundamental building blocks of AI navigation and tree traversal.


**🛠️ Tech Stack**

**⚙️ How It Works**

User enters Parent → Child relationships or uses the manual entry mode.
Defines a "Depth Limit" to control the maximum level of search.
Tree traversal.
Visual Feedback


**🎯 Objective**

To simplify the learning curve for Uninformed Search Algorithms by providing a hands-on tool that makes recursive logic and depth constraints tangible and easy to debug.
